# HouseZ
A 2D zombie survival game built in Java using Java Swing.
Fight through increasingly difficult waves of zombies using real-time movement, shooting mechanics, collision detection, and round-based progression systems.
---
## Features
- Wave-based zombie survival gameplay
- Real-time player movement and aiming
- Shooting mechanics with projectile collision detection
- Zombie AI and enemy spawning systems
- Interactive game states including round-over and game-over screens
- Audio effects and immersive gameplay systems
- Object-oriented game architecture with modular entity classes
---
## Technologies Used
- Java
- Java Swing
- Object-Oriented Programming (OOP)
- Event-Driven Programming
- Java Timers
- AudioInputStream / Clip API
---
## Core Systems
### Collision Detection
Implemented collision handling between:
- Bullets
- Zombies
- Obstacles
- Walls
- Player boundaries
### Enemy Wave System
Designed a scalable wave progression system with:
- Configurable zombie spawning
- Increasing round difficulty
- Spawn timers and round transitions
### Game Architecture
Built modular classes for:
- Player
- Zombie
- Bullet
- Obstacles
- UI/Game States
This structure improved maintainability and scalability during development.
